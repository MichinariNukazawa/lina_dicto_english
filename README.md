lina\_dicto\_english
====
\- Dictionary for Japanese / English \-

# About
lina\_dicto\_english は日本語/English変換を検索できる辞書アプリケーションです。  

Esperanto辞書 [lina\_dicto](https://github.com/MichinariNukazawa/lina_dicto) をEnglish版に改変したものです。
